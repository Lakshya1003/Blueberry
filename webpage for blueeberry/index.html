AUTONOMOUS RESILIENCE

Navigating the Unknown. Saving Lives.

Project Blueberry is an intelligent, self-reliant robotic system designed to navigate, map, and operate in complex disaster zones during total blackouts. Where humans cannot go, Blueberry leads the way.

The Challenge

Disaster zones are chaotic, unstructured, and often devoid of communication infrastructure. Traditional robotics fail when the lights go out and the signal dies.

System Architecture

Hardware Core

  • Chassis: Carbon-Fiber Reinforced Polymer
  • Drive: 4WD Independent Suspension
  • Sensors: LiDAR, Depth Cameras, IMU, Thermal
  • Compute: NVIDIA Jetson Orin Nano
  • Power: High-Density Li-Po with Solar Trickle

Software Stack

  • OS: ROS 2 (Robot Operating System)
  • AI: TensorFlow / PyTorch for Object Detection
  • Mapping: OctoMap / RTAB-Map
  • Control: PID + Model Predictive Control
  • Interface: WebSockets + React Dashboard
